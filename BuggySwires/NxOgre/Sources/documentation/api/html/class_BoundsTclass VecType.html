BoundsT

NxOgreBounds.h

A class containing the minimum and maximum bounds using a NxOgre Vector class.

Note — There are already Bounds2, Bounds3, Bounds4 typedefs already created for the
Vector2, Vector3 and Vector4 classes.

Example

 Bounds3 bounds;
 b.merge(pos1);
 b.merge(pos2);
 b.merge(pos3);
 b.merge(pos4);
 Vec3 center = b.center();
